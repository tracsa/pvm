Utilidades internas¶

Aquí se detallan algunas funciones que pueden o deben ser usadas durante el desarrollo de nuevas funcionalidades.

cacahuate.utils.get_values(execution_data)[fuente]¶

the proper and only way to get the 'values' key out of an execution document from mongo. It takes care of the transformations needed for it to work in jinja templates and other contexts where the multiplicity of answers (multiforms) is relevant.

class cacahuate.jsontypes.MultiFormDict(dict_list)[fuente]¶

A special dictionary that can take values from a list of dictionaries that share the keys, by default returns the value of the last one.

It is used to wrap the 'values' key of an execution document so it can be used in templates and other places as a normal dict that by default returns the values of the last dictionary but allows to access the rest of them via the provided functions.

all()[fuente]¶

Returns an iterator over the different dictionaries contained in this object.

dict()[fuente]¶

Return a copy of the last contained dict

get(key, default=None)[fuente]¶

Return the value for the given key. Takes the value of the last dict. If the key doesn’t exist returns default.

getlist(key, default=None)[fuente]¶

return a list of values for this key where every missing key of an underlaying dict is replaced by default.

items()[fuente]¶

Yield (key, value) pairs, from the last contained dict.

keys()[fuente]¶

Yield all the keys from the last contained dict.

values()[fuente]¶

Yield all the values from the last contained dict.
